# FREQUENCIES
This project was an assignment from an AI + Design class, centered around informing design students how to find a way to make AI work for them. As part of this project, I tested 6 different AI models, alongside some of their different tiers, with 5 different prompts. 

This field guide documents my findings from testing the different models, set against a sci-fi backdrop. I went for a dystopian/sci-fi feel inspired by the DIVISIONS era of the band Starset. This project is focused on how if you are going to use AI, you should be the one directing it, not the other way around.

Built with Claude Sonnet 4.6 and Claude Design (Claude Opus 4.7). Models change, so I have included my dates of testing and building in the About page.
